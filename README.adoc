= Electron + vue.js template

== Install

This is a project template for https://github.com/vuejs/vue-cli[vue-cli].

----
$ yarn global add vue-cli

$ vue init isuke/vuejs-electron-template my-project
$ cd my-project
$ yarn install
----

== Develop

----
$ yarn run dev
----

== Build

----
$ yarn run build
----
